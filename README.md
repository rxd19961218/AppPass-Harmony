# App Metadata Audit Skill 🚀

> **上架鸿蒙应用市场前的“模拟面试官”**。本 Skill 专注于对 App 的元数据（名称、描述、关键词等）进行自动化合规审查，提前发现违规红线，确保过审率。

## 📖 简介

在提交华为 AGC（AppGallery Connect）审核时，元数据违规是开发者最常遇到的“低级错误”。本工具通过自动化脚本扫描本地元数据文件，对照华为官方审核规范进行多维度审计，将风险消灭在提交之前。

## ✨ 核心审查能力

该 Skill 将风险分为三个等级，帮助开发者快速定位优先级：

- 🔴 **BLOCKER (阻碍)**: 绝对会被打回的违规项（如包含竞品名称、隐私链接失效）。
- 🟡 **WARNING (警告)**: 可能导致降权或审核延时的建议（如字符数接近上限、缺少关键词）。
- 🟢 **INFO (提示)**: 优化建议（如描述中未包含核心功能词）。
